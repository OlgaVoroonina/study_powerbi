# study_powerbi

**Pet-проект:** Анализ продаж интернет-магазина

**Цель проекта**
Изучение функциональности Power BI, а также разработка интерактивного дашборда для мониторинга основных бизнес-показателей.


Взяла датасет с сайта Kaggle, содержащий информацию о продажах интернет-магазина.
Исходные данные включали:
⁃ ID заказа;
⁃ дату покупки;
⁃ дату доставки;
⁃ тип доставки;
⁃ ID клиента;
⁃ имя клиента;
⁃ сегмент;
⁃ страну;
⁃ город;
⁃ штат;
⁃ почтовый индекс;
⁃ регион
⁃ сумму продажи;
⁃ количество товаров;
⁃ скидку;
⁃ прибыль.
Перед началом анализа изучила структуру данных, определила типы столбцов и проверила качество данных.

Для предварительной обработки данных использовала SQL.
В процессе работы проверила наличие:
⁃ пустых строк;
⁃ наличие дублей
⁃ аномалий в значениях

Сформировала итоговую таблицу, которая затем была импортирована в Power BI.

После обработки данных загрузила подготовленный датасет в Power BI.
Выполнила:
• настройку типов данных;
• проверку связей;
• создание модели данных;
• настройку календаря.

Для расчета KPI создала меры с использованием DAX.
⁃ Процент заказов без прибыли
⁃ Процент прибыли
⁃ Кол-во городов
⁃ Кол-во заказов
⁃ Кол-во клиентов
⁃ Кол-во штатов
⁃ Среднее время доставки
⁃ Среднее кол-во в заказе
⁃ Средний чек
⁃ Сумма прибыли
⁃ Сумма продаж
⁃ 
Создала интерактивный дашборд в Power BI.

**Используемые технологии**
• SQL — подготовка данных
• Power BI — визуализация, построение модели данных, создание интерактивных отчетов.
• DAX — расчет KPI и аналитических метрик.
